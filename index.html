FASE 1 — Fundamentos da Web

Tópicos

  • HTML5 semântico (tags, acessibilidade básica, forms).
  • CSS3: box-model, posicionamento, display, Flexbox, Grid, media queries, fontes, responsividade.
  • JavaScript básico: variáveis, tipos, operadores, condicionais, loops, funções, arrays, objetos, DOM (manipulação básica), eventos.
  • Ferramentas: VS Code, terminal básico, npm, Git & GitHub (commits, branches, push/pull).

Recursos (leitura + hands-on)

  • MDN Web Docs — referência e tutoriais sobre HTML/CSS/JS (sempre que ficar na dúvida, consulte MDN). MDN Web Docs
  • freeCodeCamp — Responsive Web Design + JavaScript (currículo com exercícios e projetos). FreeCodeCamp
  • Curso em Vídeo (Gustavo Guanabara)
  • YouTube (prática / projetos): Matheus Battisti / Hora de Codar — projetos curtos e práticos. Hora de Codar

Projetos práticos

  1. Página estática “Currículo online” (responsiva).
  2. Landing page simples com header, cards e formulário de contato (send não precisa funcionar).
  3. To-Do List vanilla JS: adicionar, remover, marcar feito + persistência no localStorage.

Critério de aceitação

  • Você consegue criar 2 páginas responsivas do zero sem copiar e colar código.
  • To-Do funcionando e publicado no GitHub Pages (ou repo com README + link).
  • Conforto com operações básicas de Git/GitHub (pushing, branching, PR simples).
